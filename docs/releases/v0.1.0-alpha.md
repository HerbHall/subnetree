# v0.1.0-alpha Release Notes

**Release Date:** February 2026
**Tag:** `v0.1.0-alpha`

## Overview

First alpha release of SubNetree -- a modular network monitoring and
management platform for HomeLabbers and small business IT.

This release provides a functional web-based network scanner with an
interactive dashboard and topology visualization. Time to First Value
is under 10 minutes with Docker.

## Features

### Network Discovery

- ICMP ping sweep and ARP scanning
- OUI manufacturer lookup (embedded database)
- Automatic device type identification
- Device lost detection background job

### Interactive Dashboard

- Device list with search, filter, sort, and pagination
- Device detail page with full device info
- Network topology visualization (React Flow)
  - Circular, hierarchical, and grid layouts
  - Status/type filters with search highlight
  - Node detail sidebar and edge info popover
  - PNG export
- Real-time WebSocket scan progress with live device feed
- Dark mode support
- First-run setup wizard with network interface selection

### Server Infrastructure

- Plugin architecture with event bus
- JWT authentication with access/refresh tokens
- RFC 7807 error responses
- Per-IP rate limiting
- Prometheus metrics at `/metrics`
- Health endpoints (`/healthz`, `/readyz`)
- OpenAPI spec with Swagger UI (dev mode)
- Backup/restore CLI commands

### Deployment

- Docker with multi-stage builds
- Docker Compose for one-command deployment
- Cross-platform binaries (Linux, macOS, Windows x amd64, arm64)

## Known Limitations

> **Note:** Pulse, Vault, Gateway, Insight, and LLM modules were fully
> implemented after v0.1.0-alpha and ship in v0.2.0. See the
> [v0.2.0 release notes](v0.2.0.md) for details.

The following remain as stubs or deferred items:

- **Agent Management (Dispatch module)**: Registered but no agents (Phase 1b)
- **Scout Agent**: Binary builds but has no production features (Phase 1b)
- **OIDC/OAuth2**: Schema ready, provider deferred to Phase 2
- **LLDP/CDP discovery**: Deferred to Phase 2
- **SNMP/mDNS/UPnP discovery**: Deferred to Phase 2

## Getting Started

### Docker (Recommended)

```bash
docker run -d --name subnetree \
  -p 8080:8080 \
  -v subnetree-data:/data \
  --cap-add NET_RAW --cap-add NET_ADMIN \
  ghcr.io/herbhall/subnetree:v0.1.0-alpha
```

Open http://localhost:8080 and complete the setup wizard.

### Binary

Download the appropriate archive for your platform from the
[GitHub Release](https://github.com/HerbHall/subnetree/releases/tag/v0.1.0-alpha).

```bash
./subnetree
```
